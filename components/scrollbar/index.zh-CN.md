## API

> 自 `pf-ui-react@0.0.9` 版本开始提供该组件。

| 参数        | 说明                                  | 类型              | 默认值  | 版本 |
| ----------- | ------------------------------------- | ----------------- | ------- | ---- |
| trigger     | 显示滚动条的时机，'none' 表示一直显示 | 'hover' \| 'none' | 'hover' |      |
| xScrollable | 是否允许横向滚动                      | 'boolean'         | 'false' |      |
| yScrollable | 是否允许竖向滚动                      | 'boolean'         | 'true'  |      |
